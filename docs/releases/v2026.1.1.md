# AtlanticWave-SDX v2026.1.1

## Definition

This platform release is defined by the Git tag `v2026.1.1` in this repository.
The tag pins the exact component versions via Git submodules.

## How to use

### Operators (recommended)

```bash
git fetch --tags
git checkout v2026.1.1

git submodule sync --recursive
git submodule update --init --recursive
git submodule status

export SDX_VERSION=v2026.1.1
docker compose pull
docker compose up -d
docker compose ps
